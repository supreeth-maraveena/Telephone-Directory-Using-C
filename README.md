# 📞 Telephone Directory Using C

## 📖 Overview
This project implements a **Telephone Directory System in C** that allows users to efficiently store, search, update, and manage contact information.

It demonstrates core programming concepts such as:
- File handling
- Data structures
- Menu-driven user interaction

---

## 🛠️ Features
- ➕ Add new contacts (name & phone number)  
- 🔍 Search for existing contacts  
- ✏️ Update contact details  
- ❌ Delete contacts  
- 📋 Display all saved contacts  
- 💾 Persistent storage using files (data is retained after program exit)

---

## 🧑‍💻 Tech Stack
- **Language:** C  
- **Compiler:** GCC / Turbo C / any standard C compiler  
- **Platform:** Cross-platform (Windows, Linux, macOS)

---

## 📦 Installation

### 1. Clone the Repository
```bash
git clone https://github.com/supreeth-maraveena/Telephone-Directory-Using-C.git
cd Telephone-Directory-Using-C
```

### 2. Compile the Program
```bash
gcc "telephone directory.c" -o telephone_directory
```

### 3. Run the Program
```bash
./telephone_directory
```

---

## ▶️ Usage
After running the program, a **menu-driven interface** will appear.

### Available Options:
- Add a new contact  
- Search for a contact  
- Update contact details  
- Delete a contact  
- Display all contacts  
- Exit  

### Example:
```
Enter your choice:
1. Add Contact
2. Search Contact
3. Update Contact
4. Delete Contact
5. Display All Contacts
6. Exit
```
